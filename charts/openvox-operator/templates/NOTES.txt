Next steps:

  1. Deploy an OpenVox stack using the openvox-stack chart:

     helm install openvox charts/openvox-stack \
       --namespace openvox --create-namespace \
       --values your-values.yaml

  2. Or create resources manually:

     kubectl apply -f config/samples/config.yaml
     kubectl apply -f config/samples/certificateauthority.yaml
     kubectl apply -f config/samples/certificate.yaml
     kubectl apply -f config/samples/server.yaml
     kubectl apply -f config/samples/pool.yaml

  3. Verify the operator is running:

     kubectl get pods -n {{ .Release.Namespace }} -l app.kubernetes.io/name={{ include "openvox-operator.name" . }}

  4. Check the operator logs:

     kubectl logs -n {{ .Release.Namespace }} -l app.kubernetes.io/name={{ include "openvox-operator.name" . }} -f
